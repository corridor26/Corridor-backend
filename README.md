# Corridor Backend

Amtrak commuter app - Delay predictions & price tracking.

## Quick Start

```bash
npm install
npm start
```

## API Endpoints

- `GET /api/trips` - List upcoming trips
- `POST /api/trips` - Create new trip
- `GET /api/trips/:id` - Get trip details
- `GET /api/search?origin=NYP&destination=WAS&date=2026-03-15`
- `GET /api/status` - Scraper status
- `GET /health` - Health check
